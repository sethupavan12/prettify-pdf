# Example Prompts

These prompts are safe to copy into an agent. Use public, synthetic, or redacted PDFs.

## Receipt Redesign

```text
Use $prettify-pdf to redesign this receipt so the total amount, receipt number, date/time, payer/payee, and transaction references are easy to find. Preserve all IDs, amounts, notes, barcode, seal, and source labels exactly.
```

## Appointment Confirmation

```text
Use $prettify-pdf to make this appointment confirmation easier to read. Put date, time window, location, applicant, application number, and required documents first. Preserve the original QR code, seal, office name, address, and all source timestamps.
```

## Invoice QA

```text
Use $prettify-pdf to QA this redesigned invoice against the original. Check invoice number, client/vendor names, dates, payment terms, line items, tax, total, amount due, logo, barcode, and any clipped or overlapping text. Tell me if it is safe to deliver.
```

## Government Form

```text
Use $prettify-pdf to redesign this government form for readability while preserving every official label, applicant detail, reference number, date, stamp, signature, QR code, and legal note exactly.
```

## Resume

```text
Use $prettify-pdf to polish this resume PDF. Improve hierarchy and readability, but do not invent or upgrade job titles, employers, dates, degrees, metrics, skills, contact details, or links.
```

## Proposal Or Report

```text
Use $prettify-pdf to redesign this report PDF. Make the objective, decision points, timeline, and next action easy to scan while preserving all scope, pricing, dates, names, disclaimers, charts, and source claims.
```
