# Changelog

All notable changes to this project will be documented in this file.

This project follows a simple `MAJOR.MINOR.PATCH` version style:

- `MAJOR`: breaking changes to skill behavior or repository layout.
- `MINOR`: new guidance, helper capabilities, docs, or eval coverage.
- `PATCH`: fixes, wording improvements, and small validation updates.

## 0.1.0 - 2026-05-14

- Initial public-ready `prettify-pdf` Agent Skill.
- Added provider-neutral `SKILL.md` for PDF redesign and QA.
- Added high-stakes preservation rules for IDs, dates, times, amounts, labels, barcodes, QR codes, seals, signatures, and source meaning.
- Added visual render comparison as a blocking quality gate.
- Added optional local `scripts/pdf_guardrails.py` helper.
- Added release docs, examples, validation script, issue templates, and GitHub Actions validation.
